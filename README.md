# Group-8-Aeris-Cave-Resort

This is a project created by 6 students of **UET Lahore** for their *Applications of Computers and Technology Lab* as a final project in collaboration with **Alzaran Software House**.

This project is a fictional holiday resort named **Aeris Cave Resort** located in the natural landscapes and caves of Cappadocia, Turkey. The website, created with HTML & CSS only, is designed to promote resort packages, cave suites, and experiences while maintaining a clean and modern style.

# Project Objectives
- Create a visually appealing resort website.
- Promote featured holiday packages.
- Practice responsive web design using CSS.
- Maintain consistent branding using colors and typography.
- Apply semantic HTML structure.

# Technologies Used
- HTML5
- CSS3
- Google Fonts
- CSS Flexbox
- Media Queries

# Design & Branding
Minimal colors and fonts have been used to achieve consistency and sleek, modern branding.

### Fonts
Heading font: "Playfair Display", serif.
CSS variable: --heading-font
Body font: "Montserrat", sans-serif.
CSS variable: --body-font
Sourced from Google Fonts.
### Colors
Name                Hex Codes    CSS Variables
Orange Primary	    #ff5e33	    --orange-primary-color
Orange Secondary	  #8b4513	    --orange-secondary-color
Dark Text          	#1b2632	    --dark-text
Light Text        	#f5f5f5    	--light-text
Blue              	#31b4b4    	--blue-1-color
### Icons
Only 3 icons are used in the contact page that are sourced from Font Awesome.
### Resposiveness
Mobile, Tablet and Desktop sizes are accomodated through media queries that specify font sizes all the way to layouts for different screen sizes.
